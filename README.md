# lie-fitness

App per monitoraggio allenamento e analisi esercizi in palestra (React + Vite, Firebase in arrivo).

**Live:** https://renato-antonio-marzia.github.io/lie-fitness/

## Sviluppo

```bash
npm install
npm run dev      # locale
npm run build    # produzione -> dist/
npm run deploy   # pubblica su gh-pages
```

## Stato

Milestone 0 = shell deployata. Backlog MVP nelle Issue (#2–#7). Firebase: vedi `.env.example` + #3.
